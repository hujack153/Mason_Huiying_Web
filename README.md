# Huiying Materials 靜態網站提案

## 如何查看

直接開啟 `index.html`，再從總覽頁選擇三種設計方向：

- `concept-a.html`：極簡建築感
- `concept-b.html`：專業工程感
- `concept-c.html`：溫暖自然感

每個提案右下角都可以快速切換版本。

所有頁面皆提供英文與繁體中文切換，預設顯示英文。使用者選擇的語言會儲存在瀏覽器中，切換提案頁面時會延續相同語言。

三個提案已進一步加入各自的品牌細節：

- A：建築比例標記、材料紋理與極簡能力數據帶
- B：工程座標、供應系統狀態與技術圖形
- C：自然材質色票、服務價值帶與品牌合作宣言

## 目前技術

目前全部使用單檔 HTML，CSS 已直接寫在各 HTML 內，不需要安裝套件、不需要資料庫，也不需要啟動伺服器。

這很適合目前「先確認視覺方向」的階段。正式內容確認後，可以繼續使用純 HTML/CSS，並部署到 GitHub Pages、Cloudflare Pages、Netlify 或一般虛擬主機。

## 正式版建議補充資料

1. 公司正式中英文名稱
2. Logo、品牌顏色與公司簡介
3. 電話、Email、地址、LINE 或其他聯絡方式
4. 產品分類、照片、用途與規格
5. 服務地區與合作流程
6. 實績或施工案例

若未來產品數量很多、需要後台自行更新，才建議改成內容管理系統或網站框架。現階段不需要先增加這些複雜度。
